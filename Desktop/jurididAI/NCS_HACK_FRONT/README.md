# JusticePath

Minimal React workspace for AI-assisted legal support.

## Stack

- Vite
- React 19
- TypeScript
- Tailwind CSS v4
- shadcn/ui
- Prompt Kit

## Structure

- `src/components` shared UI and feature parts
- `src/pages` route entrypoints
- `src/data` local app data
- `src/hooks` shared logic
- `src/types` shared types

## Commands

- `npm run dev`
- `npm run build`
- `npm run lint`

## Status

- Assistant route built.
- Other routes scaffolded.
